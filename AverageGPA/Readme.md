# 加权平均分、加权成绩计算小程序

## 使用注意事项

1. 压缩包中有三个文件，其中.c文件是源代码，.exe文件是运行程序（仅限Windows用户），.csv文件（Comma-Separated Values，CSV，有时也称为字符分隔值）用于储存自己的成绩信息。如果有兴趣在源代码基础上改动以适应自己需求，请温柔对待哈哈哈······如果仅仅使用，只需要编辑.csv文件。重点：exe文件与csv文件务必在同一目录下！！！
2. .csv文件第一行有每一列对应的信息。除了第5列和第7列，其他列与计算无关。（用处在于让自己清楚这是什么课）第5列是学分数量，第7列是对应最终分数。重点：.csv文件第一行请不要删除！！！并且不要有一个单元格有空值。
3. 计算结果包括：总课程数，总学分，加权平均分，加权平均绩点。
4. 绩点计算方法（以电子科技大学“百分制”计算方法为准，不同大学同学请自己修改计算方式，也就是Calculate函数）

    [电子科技大学教务处](http://www.jwc.uestc.edu.cn/web/News!view.action?id=1333)

5. 整体流程：将成绩按指引输入.csv文件，确保学分和最终分数两列都是数字，保存后点击exe文件执行，得到结果。
